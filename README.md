# SpringFrameworkDemo

A basic Java Maven project created for learning and practicing Spring Framework concepts.

## Tech Stack

- Java 21
- Maven
- Spring Framework
- IntelliJ IDEA



## Features

- Maven project setup
- Spring Framework practice environment
- Git/GitHub integrated

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/NirmalSilwal/SpringFrameworkDemo.git
```

2. Open in IntelliJ IDEA

3. Run:

```java
Main.java
```
